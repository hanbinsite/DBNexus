**组件尺寸**:

| 组件 | 高度 | 宽度 | 备注 |

- 4个角 (TL/TR/BL/BR): 12×12px, cursor=对应斜向
- 4条边 (T/B/L/R): 4px宽/高, cursor=对应方向
- 手柄颜色：透明，hover时 `--border` 微显

### 2.2 详细规格

**DB类型选择器**:
- 3列 grid, gap=8px, margin-bottom=20px
- 每个按钮: 宽=auto, 高=72px, 纵向排列(icon上+文字下)
- 图标: 28×28px SVG
- 文字: 11px, Medium 500
- Default色: bg=var(--bg-secondary), border=2px var(--border), color=var(--fg-secondary)
- Hover: border-color=var(--fg-muted), bg=var(--bg-hover)
- Active: bg=var(--accent-subtle), border-color=var(--accent), color=var(--accent)
- 过渡: 80ms ease-out on border-color, background-color, color

**表单字段**:
- 每个form-group: label(12px, fg-secondary, Medium 500, margin-bottom 4px) + input/select
- input: height=32px, padding=0 12px, border=1px var(--border), radius=3px, bg=var(--bg-input), fg=var(--fg-primary), 13px
- focus: border-color=var(--border-focus), box-shadow=0 0 0 3px var(--border-glow)
- placeholder: var(--fg-muted)
- 二列布局: "主机:端口" 使用 grid-template-columns: 1fr 100px, gap=12px

**SQLite特殊字段**:
- 当选择SQLite类型时，显示文件路径选择行
- 文件路径 input + [浏览] button(Secondary, 28px高)
- 隐藏"主机"和"端口"字段

**Footer**:
- height=56px, padding=0 20px, bg=var(--bg-tertiary), border-top=1px var(--border)
- 左: [测试连接] Secondary button
- 右: [取消] Ghost button + [保存] Primary button, gap=8px

### 2.3 状态矩阵

| 状态 | 视觉 |
|------|------|
| **打开对话框** | overlay fade-in 120ms, container scale(0.97→1) 200ms |
| **切换DB类型** | 按钮active状态切换 80ms, 表单字段动态切换(端口默认值变化) |
| **点击"测试连接"** | button文字→"测试中..." + disabled + spinner, 测试结果在按钮旁显示 |

| **关闭对话框** | overlay fade-out 100ms, container scale(1→0.97) |
| **ESC** | 关闭对话框（同点击✕） |

## 5. 数据浏览面板

### 5.1 面板切换

当用户在数据库树中**双击表名**，工作区切换到 Data View Panel：

```
┌──────────────────────────────────────────────────────────┐
│ [内容] [表结构] [索引] [外键]          [+] [-] [💾] [↺]  │ 36px 工具栏
│ ──────────────────────────────────────────────────────── │
│  筛选: [列 ▼] [= ▼] [值___] [应用] [清除]  排序: [列 ▼] [↕] │ 36px 过滤栏
│ ──────────────────────────────────────────────────────── │
│ ┌────────────────────────────────────────────────────┐  │
│ │ # │ id │ name          │ email           │ status  │  │ ← 表头 Sticky
│ │───┼────┼───────────────┼─────────────────┼─────────│  │
│ │ 1 │ 1  │ Alice Johnson │ alice@test.com │ ● active│  │ ← 数据行
│ │ 2 │ 2  │ Bob Smith     │ bob@test.com   │ ○ inact │  │
│ │ 3 │ 3  │ Charlie Brown │ charlie@test.. │ ● active│  │
│ └────────────────────────────────────────────────────┘  │
│  ☐ 1 ☐ 2 ☒ 3  已选: 3 │ 共 1000 条   [◀◀] [◀] 1/10 [▶] [▶▶] │ 36px
└──────────────────────────────────────────────────────────┘
```

### 5.2 工具栏

| 按钮 | 功能 |
|------|------|
| [+] 添加行 | 表格底部新增一行（绿色背景标记） |
| [-] 删除选中 | 选中行标记为删除（红色 + 删除线） |
| [💾] 保存更改 | 批量提交所有增/删/改 → `BatchEdit()` |
| [↺] 刷新 | 放弃所有未保存修改 → 重新加载 |
| [↻] 撤销更改 | `discardChanges()` — 恢复到最后一次保存状态 |

### 5.3 过滤栏

```
筛选: [列选择 ▼] [操作符 ▼] [值________] [应用] [清除]     排序: [列选择 ▼] [↕]
      ↑ 200px      ↑ 100px      ↑ 150px    ↑      ↑               ↑ 180px     ↑ 32px icon
```

- 列选择: 动态填充所有列名
- 操作符: = / != / > / < / >= / <= / LIKE / IN / IS NULL / IS NOT NULL
- 值输入: IS NULL/IS NOT NULL 时自动禁用
- [应用]: 构建 WHERE 子句并刷新表格
- [清除]: 清空WHERE → 重新加载所有数据
- 排序: 选择列 + 点击 [↕] 切换 ASC/DESC

### 5.4 数据表格详细规格

**表头**:
- height: 32px, bg=var(--table-header-bg), fg=var(--fg-secondary), 12px, Medium 500
